# 电路类课程知识图谱可视化系统

这是一个基于Web的知识图谱可视化系统，用于展示电路、模电、数电、高频电子线路、嵌入式等课程的知识点及其关联关系。

## 功能特点

- 📊 **知识图谱可视化**：使用vis-network库实现美观的交互式知识图谱
- 🔍 **节点交互**：点击任意节点查看详细信息
- 🔗 **关联展示**：自动显示选中节点与其他知识点的关联关系
- 🎨 **分类标识**：不同课程使用不同颜色区分
- 📱 **响应式设计**：支持不同屏幕尺寸

## 环境配置

### 1. 创建Conda环境

```bash
conda env create -f environment.yml
```

### 2. 激活环境

```bash
conda activate knowledge-graph
```

## 运行项目

### 启动服务器

```bash
python app.py
```

服务器将在 `http://localhost:5000` 启动。

### 访问应用

在浏览器中打开：`http://localhost:5000`

## 使用说明

1. **查看知识图谱**：页面加载后会自动显示完整的知识图谱
2. **点击节点**：点击任意知识点节点，右侧会显示该节点的详细信息
3. **查看关联**：在节点详情中可以看到：
   - 与该节点相关的其他知识点
   - 知识点之间的关联关系类型
4. **聚焦节点**：双击节点可以聚焦并放大查看
5. **交互操作**：
   - 拖拽：移动图谱位置
   - 滚轮：缩放图谱
   - 点击空白：取消选择

## 知识图谱内容

系统包含以下课程的知识点：

- **电路基础**：欧姆定律、基尔霍夫定律、戴维南定理、交流电路等
- **模拟电子技术**：二极管、三极管、运放、放大电路、滤波器等
- **数字电子技术**：逻辑门、布尔代数、组合逻辑、时序逻辑、ADC/DAC等
- **高频电子线路**：谐振电路、高频放大器、混频器、调制解调、锁相环等
- **嵌入式系统**：微控制器、ARM架构、GPIO、中断系统、串口通信等

## 技术栈

- **后端**：Flask (Python)
- **前端**：HTML5 + CSS3 + JavaScript
- **可视化**：vis-network
- **环境管理**：Conda

## 开发说明

### 添加新知识点

在 `data.py` 文件的 `get_knowledge_graph_data()` 函数中：

1. 在 `nodes` 列表中添加新节点：
```python
{'id': 新ID, 'label': '节点名称', 'group': '课程组', 'title': '详细描述'}
```

2. 在 `edges` 列表中添加关联关系：
```python
{'from': 起始节点ID, 'to': 目标节点ID, 'label': '关系类型', 'color': {'color': '#颜色代码'}}
```

### 修改样式

编辑 `static/css/style.css` 文件来自定义界面样式。

### 修改交互逻辑

编辑 `static/js/main.js` 文件来自定义交互行为。
