# 知识图谱科学构建功能 - 快速指南

## 📋 功能概述

在原有知识图谱可视化系统基础上，新增了**知识图谱科学构建**功能，包括：

1. ✅ **知识抽取**：从文本自动提取知识点和关系
2. ✅ **知识图谱嵌入**：TransE模型学习向量表示
3. ✅ **链接预测**：预测缺失的关系
4. ✅ **相关推荐**：基于相似度推荐相关概念

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 训练模型（首次使用必须）

```bash
python train_model.py
```

**预计时间**：3-5分钟

### 3. 启动应用

```bash
python app.py
```

### 4. 访问应用

打开浏览器访问：`http://localhost:5000`

## 📁 新增文件说明

| 文件 | 说明 |
|------|------|
| `kg_extraction.py` | 知识抽取模块（实体识别+关系抽取） |
| `kg_embedding.py` | TransE嵌入模型实现 |
| `kg_completion.py` | 链接预测和补全模块 |
| `train_model.py` | 模型训练脚本 |
| `使用说明-知识图谱科学构建.md` | 详细使用文档 |

## 🔧 API接口

### 知识抽取
```bash
POST /api/extract-knowledge
Body: {"text": "运算放大器是一种高增益的差分放大器"}
```

### 关系预测
```bash
GET /api/predict-relation/2/3
```

### 发现缺失链接
```bash
GET /api/find-missing-links?threshold=0.3&max=20
```

### 相关概念推荐
```bash
GET /api/recommend-related/2?top_k=10
```
